    • Linha do Tempo - Evolução dos computadores:

    (voltar)

    1. Primeira Geração (Anos 40 e 50): Surgiram os primeiros computadores eletrônicos digitais, como o ENIAC (Electronic Numerical Integrator and Computer), que ocupavam salas inteiras e eram programados manualmente por meio de chaves e cabos.

    2. Segunda Geração (Anos 50 e 60): Os computadores começaram a utilizar transistores, o que reduziu significativamente o tamanho e o consumo de energia em comparação com as válvulas eletrônicas da primeira geração. Surgiram os mainframes e os primeiros computadores de grande porte.

    3. Terceira Geração (Anos 60 e 70): A introdução dos circuitos integrados permitiu a miniaturização dos computadores, dando origem aos minicomputadores. Surgiram também os primeiros sistemas operacionais.

    4. Quarta Geração (Anos 70 até meados dos anos 90): Os microprocessadores foram desenvolvidos, permitindo a criação dos primeiros computadores pessoais (PCs), como o Altair 8800 e o IBM PC. O surgimento da Apple e da Microsoft impulsionou a popularização dos PCs.

    5. Quinta Geração (Meados dos anos 90 até os dias atuais)Houve avanços significativos em termos de velocidade, capacidade de armazenamento e conectividade. Surgiram os smartphones e tablets, que revolucionaram a computação pessoal. A computação em nuvem e a inteligência artificial também se destacam nessa era.

    A história da Internet:

    (voltar)

    O HTML, sigla para HyperText Markup Language, é a linguagem padrão utilizada para criar páginas web. Sua história remonta ao início da internet, quando Berners-Leestrog, um cientista da computação britânico, inventou a World Wide Web em 1989. Berners-Lee desenvolveu o HTML como parte do conjunto de tecnologias necessárias para compartilhar informações de forma eficaz na internet.

    Funcionamento do HTML:

    (voltar)

    O HTML, ou HyperText Markup Language, funciona como a base estrutural de uma página web. Ele utiliza uma série de elementos (também conhecidos como tags) para definir a estrutura e o conteúdo da página. Aqui está um resumo,da estrutura do HTML:

    1. Tags e Elementos:O HTML é composto por tags, que são elementos usados para marcar diferentes partes do conteúdo da página. Por exemplo, a tag "p" é usada para marcar parágrafos de texto, enquanto a tag "img" é usada para inserir imagens,a tag "strong" para deixar o texto ou palavra em negrito, "i" para deixar o texto em intálico, "u" para sublinhar o texto.

    2. Atributos:As tags HTML podem ter atributos, que são informações adicionais que especificam como um elemento deve ser exibido ou se comportar. Por exemplo, a tag img pode ter um atributo src que especifica o caminho para a imagem a ser exibida.

    Cliente x Servidor:

    (voltar)

    • Cliente: O cliente é o dispositivo ou programa que solicita serviços ou recursos de um servidor. Por exemplo, quando você acessa um site, o navegador atua como o cliente, enviando uma solicitação ao servidor do site para obter as páginas da web.

    • Servidor: O servidor é o computador ou programa que fornece os serviços ou recursos solicitados pelo cliente. Ele processa as solicitações recebidas dos clientes e envia de volta as respostas apropriadas. Por exemplo, quando você envia um e-mail, o servidor de e-mail recebe, processa e armazena sua mensagem até que o destinatário a receba.

    O que são Lista Ordenadas e Não Ordenada:

    (voltar)

    Lista Ordenada: A lista ordenada, organiza os itens da lista em formato númerico ou alfabética. Exemplo:

    1. Item 1
    2. Item 2
    3. Item 3

    Lista Não Ordenada:Uma lista não ordenada é uma lista de itens que não segue uma ordem específica. Cada item é geralmente marcado com um ponto, um quadrado ou outro símbolo.Exemplo:

    • Maçã
    • Banana
    • Uva

    Resumo:

    (voltar)

    O HTML funciona como a estrutura básica de uma página web, definindo a estrutura e o conteúdo da página por meio de tags e elementos, organizados em uma hierarquia. Ele é essencial para a criação de páginas web e é complementado por outras tecnologias para tornar as páginas mais atraentes e funcionais.
